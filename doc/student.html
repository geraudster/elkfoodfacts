2.1 Requêtes Elasticsearch

2.1.1 Insertion de données

POST /store/food
{
  "name": "Chips au vinaigre",
  "category" : "apero",
  "lipides" : 20,
  "glucides" : 10,
  "proteines" : 5
}

POST /store/food
{
  "name": "Langues piquantes",
  "category" : "confiserie",
  "lipides" : 0,
  "glucides" : 90,
  "proteines" : 5
}

Voir le contenu: 

GET /store/food/_search

2.1.2 Requêtage

GET /store/_search?q=langues

GET /store/_search
{

  "query": {
    "match": {
      "name": "langues"
    }
  }
}
